# CodeFlow Visualizer — GitHub Pages Edition

This edition is designed for the simplest possible deployment from a
Chromebook. It has no installation, terminal, Node.js, npm, build, database, or
Cloudflare requirement.

## Publish

1. Upload `index.html`, `.nojekyll`, `README.md`, and `LICENSE` to the root of
   your GitHub repository.
2. Open repository **Settings**.
3. Select **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose branch `main` and folder `/ (root)`.
6. Click **Save**.

GitHub will publish the website at:

`https://YOUR-USERNAME.github.io/codeflow-visualizer/`

## Important

Upload the individual files, not the ZIP file.
